📌 Cómo funciona el Proxy CORS

  • Proxy ON → La URL pasa por tu Cloudflare Worker antes de llegar al reproductor. Funciona en internet para streams de Bitel, Movistar, Claro, etc.
  • Proxy OFF → URL directa. Funciona solo en local o con streams que permiten CORS abierto (YouTube, CDNs públicos).
  • Si el stream da error con Proxy ON → el token de la URL puede haber expirado. Obtén una URL fresca.
